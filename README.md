# OptiQuest v0.2.1

A custom plugin for the Opticraft Hytale server.

## Features
* **Chat System:** Reworked chat system with configurable format including group name and colors.
* **Config Options:** Configure all plugin settings and messages within `config.yml`.
* **Color Codes:** Full ampersand (`&`) and Hex (`&#RRGGBB`) color support in the server config and chat.
* **Placeholder Support:** Custom placeholder support in the server config and chat.
* **Player Tracking:** Persistent player data storage such as nickname, afk and mute status within `userdata`
* **Essential Commands:**
  * `/afk` - Toggle your AFK status.
  * `/broadcast` - Broadcast a message to all online players.
  * `/list` - View all online players.
  * `/message` - Send a private message to a player.
  * `/motd` - Display the servers message of the day.
  * `/mute` - Prevent a player from speaking in chat.
  * `/near` - See which players are nearby.
  * `/nickname` - Change your display name in chat.
  * `/position` - Get a players position in the world.
  * `/rules` - Display the server rules.
