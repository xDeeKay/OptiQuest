# OptiQuest v0.1.0

A custom plugin for the OptiQuest Hytale server.

## Features
* **Color Support:** Full ampersand (`&`) and Hex (`&#RRGGBB`) color support.
* **Player Tracking:** Efficiently manages `OptiPlayer` data.
* **Essential Commands:**
    * `/afk` - Toggle your AFK status.
    * `/broadcast` - Send server-wide announcements.
    * `/list` - View all online players with AFK status.
    * `/msg` - Private messaging between players.
    * `/motd` - Display the servers message of the day.
    * `/near` - Find players in a configurable radius.
    * `/rules` - Display the server rules.
